Data conforms to SSN if:

  • Access to data is organized around Actuations, Observations and Samplings, which represent executions of Actuation, Observation and Sampling procedures, implemented by Actuators, Sensors and Samplers
  • The contents of all data fields are expressed using the appropriate RDF classes and properties from SSN, except where no such class or property exists
  • All classes and properties from SSN are used in a way consistent with the semantics declared in this specification.

SSN-compliant datasets MAY include additional non-SSN data fields and additional RDF data.
